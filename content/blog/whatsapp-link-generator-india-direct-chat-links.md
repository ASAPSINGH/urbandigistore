## 📐 Formatting Direct +91 WhatsApp Links

To format a WhatsApp link for India manually, you must join the standard API domain with the country code (+91) and the 10-digit mobile number:

$$\text{URL} = \text{"https://wa.me/91"} \mathbin{\|} \text{MobileNumber}$$

Important formatting rules:
*   **Remove Leading Zeros**: Do not include a leading `0` or prefix (e.g., `098765...` or `+9198765...`). The final number must contain exactly 12 digits (e.g., `919876543210`).
*   **Avoid Spaces or Dashes**: Do not include punctuation (e.g., `91-9876-543-210`).
